⚠ Regras do Firestore bloqueando acesso

    

      O Firebase está recusando leitura/gravação. Isso acontece quando as Regras de Segurança expiram ou estão configuradas incorretamente.


      Como corrigir:
    

    
    
      
  1. Acesse console.firebase.google.com
    2. 
      
  2. Selecione o projeto jc-dashboard-6bc08
    3. 
      
  3. Vá em Firestore Database → Regras
    4. 
      
  4. Substitua as regras pelo código abaixo e clique em Publicar
    5. 
    

    
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /jc_config/{doc} {
      allow read: if true;
      allow write: if request.auth != null;
    }
    match /jc_users/{userId}/{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
